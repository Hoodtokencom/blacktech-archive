# Blacktech Archive 🗑️

**GitHub = Trash/Archive** — Part 3 of the Dewey Knowledge System.

```
Google Drive (Brain) → Internal Drive (Body) → GitHub (Trash)
```

This repo holds clutter, old files, and misc items staged for review before permanent deletion (recycling).

## How it works

1. **Trash**: `python3 dewey_pipeline.py trash <file>` — moves clutter here
2. **Recycle**: `python3 dewey_pipeline.py recycle` — review what's here
3. **Restore or Delete**: Restore useful items, permanently delete the rest

## Part of the 3-Part System

| Part | Location | Role |
|------|----------|------|
| 🧠 Brain | Google Drive | Storefront — catalog, titles, metadata (files LOCKED) |
| 💪 Body | Internal Drive (`/media/allenai/Expansion/`) | Actual files — unlocked with KEY |
| 🗑️ Trash | GitHub (this repo) | Clutter staging — review before recycling |

Never commit sensitive data here. Passcodes stay encrypted in the vault.
